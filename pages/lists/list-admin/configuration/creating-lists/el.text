h2. Πώς δημιουργώ μια λίστα;

Πρέπει πρώτα να [[δημιουργήσετε έναν λογαριασμό λίστας->https://lists.riseup.net/www/firstpasswd/]] και στη συνέχεια να συνδεθείτε στον ακόλουθο σύνδεσμο https://lists.riseup.net. Αφότου συνδεθείτε, επιλέξτε τον σύνδεσμο [[create list -> http://lists.riseup.net/www/create_list_request]] που βρίσκεται στην κορυφή της σελίδας. Είναι σημαντικό να θυμάστε ότι οι riseup.net ηλεκτρονικού ταχυδρομείο (e-mail) λογαριασμοί είναι εντελώς ξεχαριστοί από τους riseup.net λογαριασμούς λίστας. Όταν συνδεθείτε στο lists.riseup.net, ορίστε την πλήρη διεύθυνση του ηλεκτρονικού σας ταχυδρομείου, από οποιονδήποτε παροχέα email. Δεν χρειάζεται να έχετε riseup.net λογαριασμό email για να φτιάξετε μια λίστα.
